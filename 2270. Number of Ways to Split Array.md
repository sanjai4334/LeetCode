# 2270. Number of Ways to Split Array

## Solution
```java
class Solution {
    public int waysToSplitArray(int[] nums) {
        long[] cumulative = new long[nums.length];

        long sum = 0;
        for (int i=0 ; i < nums.length ; i++) {
            sum += nums[i];
            cumulative[i] = sum;
        }
        
        int count = 0;
        for (int i = 0 ; i < nums.length - 1 ; i++) {
            if (cumulative[i] >= sum - cumulative[i]) {
                count++;
            }
        }
        return count;
    }
}
```
